Android-Workshop

Android-Programmierung für Einsteiger

Was ist das?

In diesem Kurs lernst du die Grundlagen, mit welchen du deine eigenen Apps für das Android-Betriebssystem erstellen kannst.

Was kann ich damit machen?

Wir werden eine einfache App programmieren, die dir nach dem Zufallsprinzip Vorschläge für dein Tun gibt ( Video anzeigen).
Wenn du das Prinzip verstanden hast, kannst du aber natürlich auch komplexere Apps programmieren.

Was brauche ich dafür?

  • Ein Smartphone oder Tablet mit Android.
  • Einen Laptop mit Windows, macOS oder x64-Linux.
    • Vorhandene Installation von Android Studio.
    • Vorhandene Installation von Java SE Runtime Enviroment 8.
    • Ggf. vorhandene Installation der USB-Treiber für das Telefon/Tablet unter Windows.
    • Zugriff auf Administratorrechte, falls wir etwas nachinstallieren müssen.
  • Grundlegende Englisch-Kenntnisse.
